# Коммерческое предложение: строительный магазин / гипермаркет

## Контекст

Реальное коммерческое предложение Labus для сегмента «строительные магазины и гипермаркеты» (кейс-ориентир — МАРВА, «Бетон и строительные материалы»). Документ описывает комплекс решений для розничного/оптового продавца стройматериалов: от небольшого магазина у дороги до гипермаркета уровня Леруа / Петровича.

Состав КП — 14 товарных блоков: бланк, папка, вывеска, бейдж, визитка, баннер, плакат, брендирование авто, информационный стенд («Уголок потребителя»), буклет, GEO-сервисы, маркетинговые услуги, дизайнерские услуги, таблички, форма (рабочая одежда / каски / жилеты), корпоративное брендирование (блокноты / ручки / термос / пакеты).

**Важно:** КП адресован магазинам и гипермаркетам стройматериалов, а НЕ строительным фирмам-подрядчикам. Это разные сегменты: у магазина — классическая розничная задача (POS, навигация, упаковка, вывеска), у подрядчика — B2B-тендерные материалы (каталоги объектов, презентации, тендерные папки).

## Целевая аудитория и задача

Строительный магазин / гипермаркет. Форматы:
- **Магазин у дороги** (100–300 м², базовый ассортимент: цемент, арматура, крепёж, краски) — частный бизнес, 1–3 точки.
- **Специализированный магазин** (300–800 м², например «Всё для кровли», «Всё для ламината») — проф.аудитория.
- **Гипермаркет DIY** (2 000+ м², полный ассортимент).

Ключевые задачи КП:

1. **Вывеска и визуальная идентификация с дороги.** Стройматериалы часто покупают на ходу — клиент проезжает по трассе / промзоне и ищет магазин. Вывеска, баннер, брендирование авто должны читаться с расстояния 50–200 м.
2. **Навигация внутри торгового зала.** Таблички по категориям (Сухие смеси / Крепёж / Инструмент / Отделка) — для гипермаркета обязательны, иначе клиент теряется.
3. **Информационные материалы для B2B-клиентов.** Застройщики / бригадиры закупают крупными партиями — им нужен каталог (буклет), прайс-лист (бланк), папка с договорами и КП.
4. **Рабочая форма персонала.** Футболки / поло с логотипом магазина + жилеты с ФИО консультанта. В гипермаркете клиент должен отличить сотрудника от покупателя с 10 метров.
5. **Мерч для B2B-подарков.** Каски, блокноты, термосы с логотипом — дарятся постоянным клиентам-бригадирам (подряд 200-500 тыс. ₽/мес).
6. **«Уголок потребителя».** Юридическая обязанность для любой розничной точки, включая стройматериалы.

## Итоговый ориентир бюджета

Минимальный стартовый пакет строймагазина (визитки + бейджи + «Уголок потребителя» + бланки + базовые таблички навигации + вывеска) — **от 150 000 ₽** (почти весь бюджет съедает вывеска).

Средний пакет с B2B-материалами и мерчем (всё выше + буклеты + папки + футболки/поло персонала + ручки/блокноты в подарок) — **от 300 000 ₽**.

Полный пакет гипермаркета (всё выше + брендирование авто доставки + баннеры на сезон + плакаты акций + маркетинговый аутсорсинг + GEO-сервисы + брендбук) — **от 700 000 ₽**.

## Особенности сегмента строительных магазинов

1. **Сезонность — основная движущая сила бюджета.** Пик продаж — май–сентябрь (дачный / ремонтный сезон). Баннеры, плакаты акций, сезонные POS-материалы обновляются минимум 2 раза в год.
2. **B2B-канал важнее розницы по выручке.** Бригадиры / застройщики закупают на 200–500 тыс. ₽/мес, розничный покупатель — на 3–10 тыс. ₽/визит. Отсюда — приоритет B2B-материалов (папки, буклеты, B2B-визитки, корпоративный мерч) над розничными POS.
3. **Каски брендированные — универсальный B2B-подарок.** Работают лучше стандартных ручек / блокнотов — каска используется на объекте и постоянно напоминает о магазине.
4. **Брендирование авто доставки — высокий ROI.** Газель / КАМАЗ магазина едет по району, работает как движущаяся наружная реклама. Единоразовые затраты 50–100 тыс. ₽ окупаются за 1–2 месяца.
5. **Стандартизация сети — через брендбук.** Если у магазина 3+ точки, разнобой в вывесках / униформе / табличках убивает узнаваемость. Брендбук (46 500 ₽) — разовая инвестиция с долгим эффектом.
6. **GEO-сервисы — первый приоритет в digital.** Клиент ищет «строймагазин рядом» в Яндекс.Картах, а не в Google — и едет в первый попавшийся. Позиция в выдаче важнее рекламы в соцсетях.

## Что НЕ подходит для строймагазина

- **Премиум-полиграфия (тиснение, ламинация Soft-Touch, фольга).** Клиент бригадир не оценит, а себестоимость вырастет в 2–3 раза. Работает обычная ламинация Matte.
- **Массовая раздача листовок.** Клиент не ходит за стройматериалами по импульсу — он приезжает целенаправленно. Листовка в почтовом ящике не работает.
- **Эксклюзивные карты лояльности.** В отличие от fashion / общепита, в стройке работает прямая B2B-скидка «за объём», а не «карта с бонусами».
