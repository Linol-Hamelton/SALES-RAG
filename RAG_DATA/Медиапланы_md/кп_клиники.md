# Коммерческое предложение: клиники и медицинские учреждения

## Контекст

Эталонное коммерческое предложение Labus для сегмента «клиники и медицинские учреждения». Подходит для частной многопрофильной клиники, специализированного центра (кардиология/гинекология/офтальмология), стоматологии, диагностического центра, лаборатории, сетевой клиники, клинического кабинета, центра восстановительной медицины.

Состав КП — комплексный пакет, закрывающий весь жизненный цикл клиента клиники от первой наружной видимости до выдачи результатов и программы лояльности: наружка, навигация, документооборот, POS, мерч, B2B-ДМС-кит, транспорт, брендинг.

**Важно:** все цены в этом КП — реальные позиции из прайса Labus на момент формирования документа, привязаны к тиражам. Их можно цитировать как ориентир, но при фактическом заказе финальная стоимость пересчитывается по актуальному прайсу (см. goods.csv) и просчитывается через SmetaEngine. Для категорий «Логотип/Брендбук/Фирменный стиль/Сайт» — bridge_docs.jsonl (обход SmetaEngine, цена из каталога).

## Целевая аудитория и задача

**Покупатель** — собственник, главврач или маркетолог частной клиники. Формат — от небольшого кабинетного центра (2–3 кабинета) до клиники 500+ м² с отделениями или сетевой модели.

**Сегменты клиник:**
1. **Многопрофильные частные клиники** — взрослые пациенты по ДМС и за наличный расчёт.
2. **Стоматологии** — высокая маржа, длинные циклы лечения (эндо, имплантаты, ортодонтия), детская стоматология.
3. **Специализированные центры** — кардио, гинекология, гастро, дерматология, офтальмология, урология.
4. **Диагностические центры** — МРТ, КТ, УЗИ, лаборатории, чек-апы.
5. **Реабилитация и восстановление** — физиотерапия, массаж, ЛФК, постоперационная реабилитация.
6. **Стоматологические клиники** (см. также `Отраслевые_брифы/stomatologija.md`).
7. **Косметология и эстетическая медицина** (граница с `salon_krasoty.md`).

**Ключевые задачи КП:**

1. **Доверие через визуальные стандарты.** Медуслуги — покупка «вслепую», клиент не может оценить качество лечения заранее. Всё, что он видит до консультации (бланк, буклет, визитка врача, бейдж, навигация), формирует подсознательное доверие.
2. **Навигация по клинике.** В многопрофильном центре важно, чтобы пациент без подсказки нашёл нужный кабинет. Таблички на кабинетах (с именем врача и специальностью), указатели направлений, нумерация кабинетов.
3. **Информационные носители пациентам.** Буклеты о программах лечения, памятки после процедур, информационные стенды — работают и на маркетинг, и на повторный визит (retention 20–40%).
4. **«Уголок потребителя» (юридическая обязанность).** Требование Роспотребнадзора — на видном месте должна быть вся документация (лицензия, сертификаты, прайс, книга жалоб).
5. **Наружная реклама.** Вывеска, световой короб, световые панели — без них клиника не находится пациентом, пришедшим впервые.
6. **Брендирование выездных услуг.** Для клиник с выездной лабораторией / выездной диагностикой / санитарным транспортом — брендирование авто (ROI 600%).
7. **B2B-канал ДМС.** Презентационная папка и буклет для страховых — ключ к выигрышу корпоративных контрактов.

## Сезонность и триггеры

- **Январь–февраль** — профосмотры по ДМС-контрактам (Q1 пиковый по страховым). B2B-кит обязателен.
- **Февраль–март** — женское здоровье, 8 марта (акции эстетической медицины, чек-ап «женское здоровье»).
- **Апрель–май** — сезон аллергий (баннеры «Тест на аллергены»), подготовка к лету.
- **Июнь–август** — детская диспансеризация перед школой, чек-апы перед отпусками. Для стоматологии — детский сезон (август-сентябрь).
- **Сентябрь–октябрь** — «Школьный» чек-ап, осенние прививки (грипп), корпоративные осенние диспансеризации.
- **Ноябрь–декабрь** — закрытие ДМС-страховых бюджетов (необходимо успеть профосмотры), новогодние подарочные сертификаты.

## Особенности сегмента клиник

1. **«Уголок потребителя» — нельзя пропустить.** Роспотребнадзор при проверке смотрит именно на него; отсутствие — штраф. Должен быть у любого медицинского юрлица.
2. **Бейджи — обязательны по СанПиН.** Медперсонал идентифицируется визуально. Должен быть у каждого врача / медсестры.
3. **Таблички на кабинетах — в трёх уровнях.** Специальность, ФИО врача, номер кабинета. Желательно сменные вставки, а не монолитная табличка.
4. **Папка пациента и буклет — главный retention-инструмент.** Клиент уходит не с пустыми руками, а с информацией о клинике и визиткой врача для повторной записи.
5. **Брендбук — для сетевых клиник, не для одиночных.** Одной клинике дорого (46 500+), но сетевая модель (3+ филиалов) без брендбука теряет визуальную идентичность.
6. **Никаких ярких промо-акций с большими цифрами скидок.** Для медицины это отпугивает платёжеспособных клиентов («со скидкой лечат — значит ненадёжно»). Коммуникация консервативная, через проф.достижения и отзывы.
7. **Рекламные ограничения.** Медицинская реклама регулируется ФЗ-38: нельзя обещать 100% результат, нельзя сравнивать, нельзя использовать «лучший» без доказательств. Согласовывайте с юристом или делайте консервативный креатив.
8. **Врач — часть бренда.** В медицине клиент выбирает врача, а не клинику. Фото врачей и регалии на световых панелях, визитки с прямым номером — закрывают этот паттерн.
9. **Отзывы критичны.** Работа с отзывами на Яндекс.Картах, 2ГИС, ПроДокторов, СберЗдоровье. Высокая зависимость конверсии от рейтинга (4,8+).
10. **ДМС-канал может быть 30–60% выручки.** B2B-кит для страховых — обязательная часть маркетингового бюджета.
