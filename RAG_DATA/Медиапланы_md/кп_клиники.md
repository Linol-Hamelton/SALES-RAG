# Коммерческое предложение: клиники и медицинские учреждения

## Контекст

Реальное коммерческое предложение Labus для сегмента «клиники и медицинские учреждения» (кейс-ориентир — GM-Medica). Документ описывает комплекс полиграфических, навигационных, POS-, мерч- и наружных решений для частной клиники / многопрофильного медцентра / стоматологии / диагностического центра.

Состав КП — 14 товарных блоков: визитки, бланк, папка, буклет, информационный стенд («Уголок потребителя»), таблички, брендирование (футболки / ручки / пакеты / блокноты), бейдж, вывеска, постер, световой короб, световые панели, брендирование авто, дизайнерские услуги.

**Важно:** все цены в этом КП — реальные позиции из прайса Labus на момент формирования документа, привязаны к тиражам. Их можно цитировать как ориентир, но при фактическом заказе финальная стоимость пересчитывается по актуальному прайсу (см. goods.csv).

## Целевая аудитория и задача

Клиника (частная многопрофильная / специализированная / сетевая / стоматология / диагностический центр). Формат — от небольшого кабинетного центра (2–3 кабинета) до клиники 500+ м² с отделениями.

Ключевые задачи КП:

1. **Доверие через визуальные стандарты.** Медуслуги — покупка «вслепую», клиент не может оценить качество лечения заранее. Всё, что он видит до консультации (бланк, буклет, визитка врача, бейдж, навигация), формирует подсознательное доверие.
2. **Навигация по клинике.** В многопрофильном центре важно, чтобы пациент без подсказки нашёл нужный кабинет. Таблички на кабинетах (с именем врача и специальностью), указатели направлений, нумерация кабинетов.
3. **Информационные носители пациентам.** Буклеты о программах лечения, памятки после процедур, информационные стенды — работают и на маркетинг, и на повторный визит.
4. **«Уголок потребителя» (юридическая обязанность).** Требование Роспотребнадзора — на видном месте должна быть вся документация (лицензия, сертификаты, прайс, книга жалоб).
5. **Наружная реклама.** Вывеска, световой короб, световые панели — без них клиника не находится пациентом, пришедшим впервые.
6. **Брендирование выездных услуг.** Для клиник с выездной лабораторией / выездной диагностикой — брендирование авто.

Для клиники — визитка врача (с прямым номером для записи) + визитка клиники (с QR-кодом на онлайн-запись).

Минимальный стартовый пакет клиники (визитки + бланки + бейджи + «Уголок потребителя» + таблички на кабинеты из композита) — **от 50 000 ₽**.

## Особенности сегмента клиник

1. **«Уголок потребителя» — нельзя пропустить.** Роспотребнадзор при проверке смотрит именно на него; отсутствие — штраф. Должен быть у любого медицинского юрлица.
2. **Бейджи — обязательны по СанПиН.** Медперсонал идентифицируется визуально. Должен быть у каждого врача / медсестры.
3. **Таблички на кабинетах — в трёх уровнях.** Специальность (Хирург / Терапевт), ФИО врача, номер кабинета. Желательно менять ФИО-часть при смене врача (отсюда — сменные вставки, а не монолитная табличка).
4. **Папка пациента и буклет — главный retention-инструмент.** Клиент уходит не с пустыми руками, а с информацией о клинике и визиткой врача для повторной записи.
5. **Брендбук — для сетевых клиник, не для одиночных.** Одной клинике дорого (от 46 500 ₽), но сетевая модель (3+ филиалов) без брендбука теряет визуальную идентичность.
6. **Никаких ярких промо-акций с большими цифрами скидок.** Для медицины это отпугивает платёжеспособных клиентов («со скидкой лечат — значит ненадёжно»). Коммуникация консервативная, через проф.достижения и отзывы.
